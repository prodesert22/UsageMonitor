# v0.1.0

Initial release. Linux port of [CodexBar](https://github.com/steipete/CodexBar)
as a Rust library + CLI for monitoring AI service usage from the terminal.

## Changes

- `usage-monitor-core` library: usage models (rate windows, costs, credits,
  plans), provider trait and registry.
- `claude` provider: Claude Pro/Max subscription usage via Claude Code OAuth
  credentials (`~/.claude/.credentials.json`), with automatic token refresh.
  Shows 5h session, weekly, and per-model weekly rate limit windows.
- `anthropic` provider: Anthropic Admin API (usage report, cost report, rate
  limit probe).
- `openai` provider: rate limit headers, organization costs and token usage.
- `usage-monitor-cli`: `list` and `fetch <provider>` commands, text and
  `--json` output.

## Build from source

```bash
git checkout a715090
cargo build --release
./target/release/usage-monitor-cli fetch claude
```
