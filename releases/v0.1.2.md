# v0.1.2

Adds the `codex` provider for ChatGPT plan Codex usage.

## Changes

- `codex` provider: reads Codex CLI OAuth credentials
  (`$CODEX_HOME/auth.json` or `~/.codex/auth.json`) and queries the ChatGPT
  backend usage endpoint. Shows 5h session and weekly rate limit windows,
  plan type, and credits.
- Automatic token refresh on rejection, persisting rotated tokens back to
  `auth.json` while preserving unknown fields.

## Build from source

```bash
cargo build --release
./target/release/usage-monitor-cli fetch codex
```
