# v0.6.1

Polishes the repository presentation after the desktop-widget release and keeps
the KDE plasmoid on Plasma's stock system-monitor icon.

## Added

- Project logo asset in [`assets/UsageMonitor.png`](../assets/UsageMonitor.png).
- Centered project logo at the top of the README.
- README badges for CI, license, latest GitHub release, Rust 2024 edition, last
  commit, and Linux platform support.
- GitHub Actions CI workflow with Rust formatting, Clippy, Rust tests, Ruff, and
  widget Python tests.

## Changed

- KDE plasmoid metadata, About dialog, and panel representation use the default
  `utilities-system-monitor` icon provided by the active Plasma icon theme.
- Workspace crate versions and KDE plasmoid metadata were bumped to `0.6.1`.

## Build from source

```bash
cargo build --release
# binary at target/release/usage-monitor-cli
```
