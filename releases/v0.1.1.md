# v0.1.1

Adds color to the CLI usage bars.

## Changes

- Colored usage bars in the CLI: green below 70%, yellow from 70%, red
  from 90%. Color is disabled when stdout is not a terminal or `NO_COLOR`
  is set, so `--json` and piped output stay clean.

## Build from source

```bash
cargo build --release
./target/release/usage-monitor-cli fetch claude
```
